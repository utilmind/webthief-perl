Welcome
----------------------------------------------------
With the help of scripts of this package and 'HTTPGET.LIB' library,
Perl programmer get an opportunity to receive files and their data
from web via HTTP protocol. At reception of a file, programmer should
specify just usual url address.


HTTLGET.LIB - How to use ?
----------------------------------------------------
This library contains follwing routines:

 1. HTTP_GET - This routine divides query string (url address) and
               receives data being on url address in web. After receiving,
               this routine will divide data to header and body.

 2. HTTP_GET_HEADER - This routine divides query string (url address) and
                      receives the header of data being on url address in web.


For more information, see the HTTPGET.LIB file or
WEBTHIEF.CGI and GETHEADER.CGI for examples.
